Zhong, "[Millimeter-wave MIMO Transmission for FBMC Systems with Lens Antenna Arrays](https://doi.org/10.1109/LSP.2025.3547268)," in IEEE Signal Processing Letters, doi: 10.1109/LSP.2025.3547268.
